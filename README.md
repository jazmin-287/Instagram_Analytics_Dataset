# Instagram_Analytics_Dataset
# Проект: Анализ данных в Instagram
# 1. Резюме проекта
   Данный проект был создан для анализа контента, ER, охватов, типов публикаций, категорий в Instagram. Данные были предоставлены для сдачи экзамена в AlmaU.
# 2. Бизнес-вопросы
1. Какие категории контента дают самый высокий ER (Engagement Rate)?
2. Какие типы контента (видео, карусель, фото) дают лучший охват?
3. В какое время публикации дают максимальный ER и охват?
4. Какой тип контента лучше работает в выходные и в будние дни?
5. В какие дни недели и в какое время публикации дают максимальный ER_on_Reach, и как это можно использовать для оптимизации расписания постинга?
6. Влияет ли количество хэштегов на эффективность публикации (ER, охват, сохранения)?
7. Есть ли связь между длиной текста и вовлечённостью?
8. Как распределяются источники трафика, и какие из них приносят наибольший вклад в общий охват контента?
9. Как изменяется ER_on_Reach со временем, и какие периоды показывают рост или падение эффективности контента?
10. Какие типы контента приводят к наибольшему приросту подписчиков, и какие форматы стоит масштабировать?
